monix.eval.Coeval

EvalAlways

Related Doc: package Coeval

final case class EvalAlways[+A](f: () ⇒ A) extends Coeval[A] with Product with Serializable

Constructs a lazy Coeval instance.

This type can be used for "lazy" values. In some sense it is equivalent to using a Function0 value.
